# Claim API

Verify your Solana Agentbot token balance and claim free agent credits. Token holders are assigned to a tier based on their balance, and credits are granted once per wallet.

## Claim credits

```http
POST /api/claim
```

Verifies the caller's Solana token balance and grants credits based on the matching tier. Each wallet can claim once. If no account exists for the provided email, one is created automatically.

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `solana_address` or `solanaAddress` | string | Yes | Solana wallet address (base58-encoded, 32–44 characters). Both snake_case and camelCase are accepted. |
| `email` | string | Yes | Email address to associate with the claim. If no account exists, one is created with the `free` plan. |

### Example request

```json
{
  "solanaAddress": "7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU",
  "email": "user@example.com"
}
```

You can also use the snake_case form:

```json
{
  "solana_address": "7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU",
  "email": "user@example.com"
}
```

### Credit tiers

Credits are determined by your Agentbot token balance at the time of the claim:

| Tier | Minimum balance | Credits granted |
|------|----------------|----------------|
| Whale | 100,000 | 500 |
| Builder | 10,000 | 150 |
| Holder | 1,000 | 50 |

The first matching tier is used (highest balance threshold first).

### Response

```json
{
  "success": true,
  "tier": "Builder 🔧",
  "credits": 150,
  "balance": 15000,
  "message": "Claimed 150 free agent credits as Builder 🔧!"
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | `true` when credits were granted |
| `tier` | string | The tier label assigned to the wallet |
| `credits` | number | Number of credits granted |
| `balance` | number | The token balance that was verified |
| `message` | string | Human-readable confirmation |

### Errors

| Code | Description |
|------|-------------|
| 400 | Missing `solanaAddress` (or `solana_address`) or `email` |
| 400 | Invalid Solana address format |
| 400 | Insufficient balance. Response includes `balance`, `minimumRequired`, and a `message` explaining the minimum threshold. |
| 409 | Wallet has already claimed. Response includes `claimedAt` (timestamp) and `credits` (amount previously granted). |
| 500 | Internal server error |
| 502 | Could not verify token balance via Solana RPC |

### Error response (already claimed)

```json
{
  "error": "Already claimed",
  "claimedAt": "2026-04-10T12:00:00.000Z",
  "credits": 150
}
```

### Error response (insufficient balance)

```json
{
  "error": "Insufficient balance",
  "balance": 500,
  "minimumRequired": 1000,
  "message": "You need at least 1,000 Agentbot tokens to claim credits."
}
```

## Check eligibility

```http
GET /api/claim
```

Checks whether a Solana address is eligible to claim credits without actually claiming. Returns the current token balance, matching tier, and whether the wallet has already claimed.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `address` | string | Yes | Solana wallet address to check |

### Example request

```bash
curl "https://agentbot.sh/api/claim?address=7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU"
```

### Response

```json
{
  "balance": 15000,
  "eligible": true,
  "tier": "Builder 🔧",
  "credits": 150,
  "alreadyClaimed": false,
  "tiers": [
    { "minBalance": 100000, "credits": 500, "label": "Whale 🐋", "current": false },
    { "minBalance": 10000, "credits": 150, "label": "Builder 🔧", "current": true },
    { "minBalance": 1000, "credits": 50, "label": "Holder 💎", "current": false }
  ]
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `balance` | number | Current Agentbot token balance for the address |
| `eligible` | boolean | `true` if the wallet qualifies for a tier and has not already claimed |
| `tier` | string \| null | Matching tier label, or `null` if balance is below the minimum threshold |
| `credits` | number | Credits that would be granted, or `0` if not eligible |
| `alreadyClaimed` | boolean | `true` if this wallet has already claimed credits |
| `tiers` | array | All available tiers with a `current` flag indicating the wallet's tier |
| `tiers[].minBalance` | number | Minimum token balance for this tier |
| `tiers[].credits` | number | Credits granted at this tier |
| `tiers[].label` | string | Tier display name |
| `tiers[].current` | boolean | `true` if this is the wallet's current tier |

### Errors

| Code | Description |
|------|-------------|
| 400 | Missing `address` query parameter |
| 502 | Could not verify balance via Solana RPC |
