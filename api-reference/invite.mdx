# Invite API

Generate and validate invite codes for gating access to the Agentbot platform. These endpoints are backend-only and do not require authentication.

<Note>Invite codes are stored in memory and are not persisted across server restarts. This is a demo implementation — production deployments should use database-backed storage.</Note>

## Generate invite code

```http
POST /api/invite/generate
```

Generates a new invite code. No authentication required.

### Response

```json
{
  "code": "a1b2c3d4e5f6"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `code` | string | A 12-character hex string generated from 6 random bytes |

## Validate invite code

```http
POST /api/invite/validate
```

Validates and consumes an invite code. Once validated, the code is marked as used and cannot be reused. No authentication required.

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `code` | string | Yes | The invite code to validate |

### Response (valid)

```json
{
  "valid": true
}
```

### Response (invalid)

```json
{
  "valid": false
}
```

### Errors

| Code | Description |
|------|-------------|
| 400 | Code is missing, invalid, or already used |
