# Skills API

Use skill endpoints to extend your agents with specialized capabilities.

## Base URL

```
https://agentbot.raveculture.xyz/api/skills
```

## Available Skills

| Skill | Endpoint | Description |
|-------|----------|-------------|
| Visual Synthesizer | `/visual-synthesizer` | Generate artwork with Stable Diffusion XL |
| Track Archaeologist | `/track-archaeologist` | Search catalog for similar tracks |
| Setlist Oracle | `/setlist-oracle` | Build DJ sets with BPM/key analysis |
| Groupie Manager | `/groupie-manager` | Fan CRM and segmentation |
| Royalty Tracker | `/royalty-tracker` | Track streaming royalties |
| Demo Submitter | `/demo-submitter` | Submit demos to Base FM |
| Event Ticketing | `/event-ticketing` | Sell tickets with x402 USDC |
| Event Scheduler | `/event-scheduler` | Schedule events across channels |
| Venue Finder | `/venue-finder` | Find venues worldwide |
| Festival Finder | `/festival-finder` | Discover festivals globally |

## List Skills

```http
GET /api/skills
```

Returns all available skills in the marketplace.

## Use a Skill

```http
POST /api/skills/{skill-name}
```

### Visual Synthesizer

```json
{
  "prompt": "dark techno album cover",
  "style": "industrial"
}
```

### Track Archaeologist

```json
{
  "action": "search",
  "bpm": 128,
  "genre": "techno",
  "mood": "dark"
}
```

### Setlist Oracle

```json
{
  "action": "build",
  "genre": "house",
  "duration": 120
}
```

### Groupie Manager

```json
{
  "action": "segment"
}
```

### Royalty Tracker

```json
{
  "action": "total"
}
```

### Demo Submitter

```json
{
  "action": "submit",
  "title": "My Track",
  "artist": "My Name"
}
```

### Event Ticketing

```json
{
  "action": "purchase",
  "eventId": "e1",
  "email": "fan@example.com",
  "tier": "vip"
}
```

### Event Scheduler

```json
{
  "action": "schedule",
  "title": "Newsletter",
  "date": "2026-03-20",
  "time": "18:00",
  "channels": ["telegram", "email"]
}
```

### Venue Finder

```json
{
  "action": "search",
  "city": "London",
  "type": "underground",
  "capacity": 200,
  "maxPrice": 500
}
```

### Festival Finder

```json
{
  "action": "search",
  "genre": "techno",
  "country": "UK",
  "maxPrice": 350
}
```

## Response Format

```json
{
  "success": true,
  // skill-specific response data
}
```

## Errors

| Code | Description |
|------|-------------|
| 400 | Invalid request parameters |
| 404 | Skill not found |
| 500 | Internal error |
