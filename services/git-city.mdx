# Git City Integration

**Analyze any GitHub repository and visualize it as a 3D city.** Powered by git-city, the platform provides deep analytics on repositories, commits, contributors, and more.

## Features

### Repository Analysis
- **Commits** — View total commit count and recent commit history
- **Contributors** — Track unique contributors and their activity
- **Stars & Forks** — Monitor repository popularity
- **Topics & License** — Understand repository metadata

### 3D Visualization
- Repositories visualized as interactive 3D cities
- Explore repositories visually in the dashboard

### API Integration

```typescript
// Analyze a repository
const response = await fetch('/api/git-city', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ repoUrl: 'https://github.com/owner/repo' })
});

const data = await response.json();
// Returns: { repository, stats, commits }
```

### Endpoint

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/git-city` | Analyze GitHub repository |

### Request Body

```json
{
  "repoUrl": "https://github.com/owner/repo"
}
```

## Access

Visit `/dashboard/git-city` on the platform to access the Git City analyzer. Enter any public GitHub repository URL to get detailed analytics and visualization.
